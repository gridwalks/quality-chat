Security & access

Sign-in is handled by Netlify Identity with invite-only registration. Admins can grant the admin role for elevated access. The chat uses a server-side API key—your key never runs in the browser.

FAQ

Does it store our data?

Chat requests are sent server-side with your API key. You can disable or replace the usage logger, or route logs to your preferred store.

Can we import our SOPs?

This starter uses a lightweight JSON FAQ. We can connect a document store if you’d like.
